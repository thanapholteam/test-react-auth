# React Authentication

React authentication with forgot password OTP

## ⚙️ Setup

Clone the Repo

```bash
git clone https://github.com/thanapholteam/test-react-auth.git
cd test-react-auth
```

Install dependencies

```sh
yarn
```

## Start

### 🛠️ Development

```sh
yarn dev
```

### 🏭 Production

Build for production

```sh
yarn build
```
